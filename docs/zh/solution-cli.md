# CLI

本项目默认已经安装 Drupal 的命令行工具：[Drush,Drupal Console](https://www.drupal.org/docs/user_guide/en/install-tools.html#s-what-are-command-line-tools)

使用 SSH 连接服务器，便可以直接使用：

```
# Drush 所有命令行
dcg  
drush  
php-parse 
psysh  
release  
robo  
security-checker  
var-dump-server

# Drupal Console 所有命令行
drupal  
php-parse  
psysh  
var-dump-server
```
