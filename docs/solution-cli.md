# CLI

Drupal provided two sets of CLI tools: drush and Drupal_ console

drupal_console contains the following commands
```
drupal  
php-parse  
psysh  
var-dump-server
```

drush contains the following commands
```
dcg  
drush  
php-parse 
psysh  
release  
robo  
security-checker  
var-dump-server
```
